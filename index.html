HTML

  • HTML stands for Hypertext Markup Language and used to create pages and web apps
  • HTML uses tags and elements to structure a web page.
  • The head element contains info about the web page or also known as Metadata
  • The body element represents the visible content shown to the user
  • header, footer, div, span are some of the section elements
  • h1, h2, h3.. are used to represent header and p tags for paragraph
  • img tag can add images into a website
  • a tag is used to add link onto a webpage with href attribute to specify the destination of the link
  • br is used to add a break line

GIT

  • GIT is a version control system and GitHub is a version control cloud service
  • Git Bash is used to execute GIT commands
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status: used to check the current working branch
  • git pull origin main: used to pull main branch to confirm local repo is uptodate
  • git checkout main: switches current working branch to main
  • git add -A: staging changed files and saving all changes
  • git commit: used to commit changes to current working branch
  • git push : used to upload local repository content to a remote repository.
